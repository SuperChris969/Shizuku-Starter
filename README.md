Shizuku Starter is a lightweight module designed to automate the activation of the Shizuku server via ADB shell. It provides a convenient "Action" button within supported managers (like AX Manager) to launch Shizuku without needing to manually run terminal commands or connect to a PC every time you reboot.

Features
• One-Tap Activation: Quickly start the Shizuku server using the integrated action script.
• Auto-Detection: Checks if Shizuku is installed and if the required start script exists.
• Safety Checks: Verifies Android API compatibility (Requires Android 8.0+).
• Status Logging: Provides real-time feedback during installation and activation.

Requirements
• Android Version: Android 8.0 or higher (API 26+).
• Shizuku App: Must be installed on the device (Play Store or GitHub).
• AX Manager: Required to trigger the action.sh script via the UI.
